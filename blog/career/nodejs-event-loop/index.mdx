The following code can be used to clearly see how Node.js executes asynchronous code in the synchronous workflow.

JavaScript Code:
```javascript
console.log("1");

setTimeout(() => console.log("macrotask"), 0);

Promise.resolve().then(() => console.log("microtask"));

console.log("2");
```

Expected Output:
```javascript
1
2
microtask
macrotask
```
